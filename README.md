# Simple Cube

단순한 카피 프로젝트가 아닌 새로 개발해보고 싶어서 시작한 간단한 큐브 게임 사이트 프로젝트

![image](https://github.com/Rowan-Hyeho-Song/SimpleCube/assets/158243861/85812524-da1b-4689-b5a0-ba9fb73c1789)


## 기능
1. Theme Change : 테마를 변경합니다. (라이트/다크)
   
   ![image](https://github.com/Rowan-Hyeho-Song/SimpleCube/assets/158243861/d009522d-1824-4565-8594-e2f27ae453c6)
   ![image](https://github.com/Rowan-Hyeho-Song/SimpleCube/assets/158243861/9b0875b4-e680-4f78-a97b-2578e7c831a1)

2. Run Mode Change : 플레이 모드를 변경합니다. (플레이/풀이)

   ![image](https://github.com/Rowan-Hyeho-Song/SimpleCube/assets/158243861/e66b2d0f-26b8-40f4-aa50-73b7e3f8536c)

   - 플레이 모드
     ![image](https://github.com/Rowan-Hyeho-Song/SimpleCube/assets/158243861/1ab96e4c-49e0-4a93-9ddf-91ee18829a23)

   - 풀이 모드(작업중)
     ![image](https://github.com/Rowan-Hyeho-Song/SimpleCube/assets/158243861/c56eea26-9db4-4068-bf40-346d80e42ce3)

3. Cube Type Setting : 큐브의 종류를 선택합니다. (2x2x2/3x3x3)

   ![image](https://github.com/Rowan-Hyeho-Song/SimpleCube/assets/158243861/159d0e25-32c0-4edc-a740-3828ad099e85)

   - 2x2x2
   ![image](https://github.com/Rowan-Hyeho-Song/SimpleCube/assets/158243861/915d91fa-a200-417c-b673-66fd89f86e6c)

   - 3x3x3
   ![image](https://github.com/Rowan-Hyeho-Song/SimpleCube/assets/158243861/1ab96e4c-49e0-4a93-9ddf-91ee18829a23)

4. Add penalty : 패널티를 추가합니다. (블라인드/횟수제한/시간제한)

   ![image](https://github.com/Rowan-Hyeho-Song/SimpleCube/assets/158243861/b084fa3f-ec6c-44dc-8075-b064928ec68c)

   - 블라인드

   - 시간제한
